What is WSL??

Windows Subsystem for Linux is commonly known has WSL. It allows the user to run their favorite Linux command line tools, Utilities, Applications natively on Windows. without the overhead of a traditional virtual machine or dual-boot setup. You can also access the windows files from that environment and run windows executables.
