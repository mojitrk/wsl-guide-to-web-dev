Linuxify Windows
Introduction Rationale Installation Guide



Prerequisites

1

Windows 10 (version 2004) or Windows 11

Press Windows Key + R and type in "winver" to get your current Windows version.

2

Windows Subsystem for Linux

Enable Windows Subsystem for Linux in the Windows Features panel.

3

Virtual Machine Platform

Enable Virtual Machine Platform in the Windows Features panel.

4

Hyper-V

Enable Hyper-V in the Windows Features panel.

Simplified Installation

1

Open Powershell

Make sure you are running it as Administrator.

2

$ wsl --install

For further clarification, this simplified installer automatically download and installs the Linux kernel and also sets the WSL version to 2, download and install Ubuntu Linux.

Configuration

1

Open Ubuntu Linux

This will open a plain command prompt window.

2

Account Setup

Enter a new username and password. (note: the password is invisible and won't show up as asterisks when you type it in)

3

Update to the Latest Version

Run "$ sudo apt update" and "$ sudo apt upgrade" to update all of Ubuntu's packages to the latest version.

Installing WSLg

1

Open Powershell

Make sure it is running as Administrator.

2

$ wsl --update

This will automatically update WSL to the latest version, including support for graphical apps that can run on the X server.

3

Open WSL Apps

Any apps with a GUI installed in WSL will show up in the Windows Start Menu.
